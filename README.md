### Num Py Examples
##### Where 
      a = np.arange(5,10)
      np.where(a < 8)       # tell me where in a, entries are < 8
      (array([0, 1, 2]),)   # answer: entries indexed by 0, 1, 2
      
      a[np.where(a < 8)]    # Pick the indices values from array 
      array([5, 6, 7])      # selects from a entries 0, 1, 2
